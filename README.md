# SupplyWok

SupplyWok is a web-based platform designed to optimize operational and supply processes for *chifa* restaurants and their suppliers. It integrates inventory management, order coordination, demand projection, and IoT-based monitoring into a single system.

All project documentation is organized in the `/documentation` directory.

## 🌐 Features

* Inventory and stock management
* Order tracking between restaurants and suppliers
* Demand forecasting
* Alerts and monitoring (IoT integration)
* Role-based access (restaurant / supplier)
